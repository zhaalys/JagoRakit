FAQ

Bagaimana cara memesan?
+
Pilih paket > kirim data melalui WhatsApp > kami proses.
Metode pembayaran?
+
Transfer bank / e-wallet / pembayaran via marketplace (tergantung promosi).
Berapa lama proses pengerjaan?
+
Umumnya 1–3 hari kerja, tergantung paket & antrian.
Apa ada garansi?
+
Kami menyediakan revisi sesuai paket. Revisi tambahan berbayar.
Butuh data apa?
+
CV lama, riwayat pendidikan, pengalaman, portfolio (jika ada), foto profesional (opsional).
Apakah file bisa diedit ulang?
+
Ya, kami kirim file dalam format Word/Canva (sesuai paket) agar bisa diedit kembali.
Bisa request desain custom?
+
Bisa, tinggal kirim referensi desain atau konsep yang diinginkan.
Apa bedanya paket Basic & Premium?
+
Basic untuk desain sederhana, Premium mencakup desain lebih kreatif & revisi lebih banyak.
Apakah ada contoh desain?
+
Ya, silakan lihat di halaman produk kami atau minta katalog via WhatsApp.
Apakah semua data saya aman?
+
Data Anda dijamin aman dan tidak akan dibagikan ke pihak lain.
Bisa revisi berapa kali?
+
Tergantung paket: Basic 1x revisi, Premium 3x revisi, Unlimited dengan biaya tambahan.
Bisa buat portofolio juga?
+
Ya, selain CV kami juga bisa membuat portofolio profesional & modern.
Bisa pakai bahasa Inggris?
+
Bisa, kami menyediakan opsi CV bilingual (Indonesia & Inggris).
Bisa cetak fisik?
+
Untuk saat ini hanya file digital. Cetak bisa dilakukan di percetakan terdekat.
File dikirim dalam format apa?
+
PDF untuk hasil akhir, Word/Canva/AI tergantung paket yang dipilih.
Apa ada diskon untuk mahasiswa?
+
Ya, ada promo khusus mahasiswa/fresh graduate. Hubungi admin untuk detail.
Bisa pesan untuk banyak orang?
+
Bisa, misalnya untuk teman sekelas atau tim. Ada harga khusus untuk order grup.
Apa ada paket langganan?
+
Ya, untuk HR/recruiter yang butuh banyak desain CV bisa ambil paket corporate.
Bisa tambah foto di CV?
+
Bisa, foto bisa disertakan atau dibiarkan polos sesuai preferensi Anda.
Apakah desain sesuai ATS?
+
Ya, kami menyediakan desain ATS-friendly agar mudah terbaca sistem rekrutmen.
Bisa revisi setelah jadi?
+
Bisa, selama masih dalam masa revisi sesuai paket. Lewat masa revisi ada biaya tambahan.
Apakah bisa buat cover letter?
+
Ya, kami juga menyediakan jasa pembuatan cover letter profesional.
Bisa pakai desain untuk LinkedIn?
+
Bisa, kami juga bisa bantu optimasi tampilan profil LinkedIn Anda.
Apakah hasil bisa dipakai untuk beasiswa?
+
Ya, desain CV bisa disesuaikan untuk keperluan akademik/beasiswa.
Bagaimana jika saya belum punya pengalaman kerja?
+
Kami bantu tonjolkan pendidikan, skill, & pengalaman organisasi agar CV tetap menarik.
